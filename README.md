# myvn - No-Code GUI-Конструктор Визуальных Новелл (Яндекс Игры)

Визуальный десктопный редактор на **Python (PySide6)**, позволяющий собирать интерактивные новеллы с диалогами, выборами и спрайтами без навыков программирования.
На выходе генерируется легкая сборка игры на чистом **JavaScript (Vanilla JS)**, адаптированная для экранов мобильных телефонов и ПК и готовая к публикации на **Яндекс Игры**.

## Функционал и особенности:
1. **Управление сценами и диалогами**:
   - Создание, удаление и переименование игровых сцен.
   - Настройка фона для каждой сцены (картинка или цвет).
   - Список реплик с возможностью указывать спикера, текст, изображение персонажа (спрайт) и его положение (слева, по центру, справа).
   - Возможность менять порядок реплик (вверх/вниз).
   - Поддержка ветвления сюжета (кнопки выбора на другие сцены) или автоматический переход (jump) по окончанию диалога.
2. **Встроенная монетизация для Яндекс Игр**:
   - Автоматический вызов полноэкранной рекламы (Interstitial) с настраиваемым интервалом (по умолчанию 3 минуты).
   - Логика паузы игры и заглушения звуков во время показа рекламы.
   - Поддержка In-App покупок (отключение рекламы за реальные деньги / Ян.Валюту).
   - Автоматическая проверка покупок при старте игры через SDK.
   - Автоматический рендеринг кнопки отключения рекламы (в главном меню и HUD игры) при указании ID покупки.
3. **Локальная песочница (Mock SDK)**:
   - Если вы открываете игру на локальном компьютере (без подключения Яндекс SDK), игра не упадет с ошибкой. Скрипт переключится в режим симуляции и покажет интерактивное окно полноэкранной рекламы и платежей для проверки логики.

## Как запустить редактор:

1. **Создайте и активируйте виртуальное окружение**:
   ```bash
   # Перейдите в папку с проектом
   cd F:\myvn
   # Активируйте виртуальное окружение
   .venv\Scripts\activate
   ```

2. **Запустите редактор**:
   ```bash
   python editor.py
   ```

## Формат Экспорта проекта

При нажатии кнопки **«Экспорт в HTML5»** в папке экспорта создается:
- `index.html` — основная веб-страница игры (с подключенным SDK v2).
- `style.css` — адаптивная верстка игрового экрана с эффектом стеклянного размытия (glassmorphism).
- `game.js` — движок воспроизведения новеллы.
- `yandex_sdk.js` — модуль обертки для вызова функций Яндекса.
- `story.js` — сгенерированные данные вашей новеллы в формате JSON.
- Папка `assets/` — сюда автоматически копируются все используемые в новелле картинки фонов и спрайты персонажей с преобразованием путей в относительные.
- **ZIP-архив `{Имя}_yandex.zip`** — готовый архив для загрузки в консоль разработчика Яндекс Игр.

## Настройки в консоли Яндекс Игр
Для работы монетизации:
1. Зарегистрируйтесь в консоли разработчика [Яндекс Игры](https://games.yandex.ru/dev).
2. Загрузите полученный ZIP-архив из папки экспорта.
3. Перейдите во вкладку **Покупки** в вашей консоли и создайте нерасходуемую покупку с ID, совпадающим с ID покупки в настройках редактора (например, `no_ads_30rub`). Укажите стоимость и описание.
4. Отправьте игру на модерацию.
